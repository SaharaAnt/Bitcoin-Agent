# 📐 Ahr999 囤币指数：当前价格贵不贵？

> **读完本文你将理解**：当你看到"Ahr999 = 0.82 (🟡 定投区间)"时，这个数字是怎么算出来的，以及它究竟意味着什么。

---

## 1. 一个核心问题：比特币现在贵吗？

价格 $80,000 是贵还是便宜？
- 如果 2021 年你说贵，你是对的（随后暴跌）
- 如果 2023 年你说贵，你是错的（随后翻倍）

Ahr999 解决了这个问题：它不是看**绝对价格**，而是看**相对于两个基准的位置**。

---

## 2. 两个基准是什么？

```mermaid
flowchart TD
    PRICE[当前 BTC 市场价格] 
    
    PRICE --> B1[基准一：200日均线\n200 Day Moving Average\n过去200天的平均价格]
    PRICE --> B2[基准二：指数增长预期价格\n基于比特币存在天数的数学模型]

    B1 --> RATIO1[价格 ÷ 200日均线\n= 当前是否偏离历史均值？]
    B2 --> RATIO2[价格 ÷ 预期价格\n= 当前是否符合长期增长模型？]

    RATIO1 & RATIO2 --> AHR[Ahr999 = 比率1 × 比率2]
```

---

## 3. 「指数增长预期价格」是什么

这是 Ahr999 指标最独特的部分。它基于一个观察：

> **比特币诞生至今越来越多天，其「公平价值」理论上也随着时间指数增长。**

计算公式：
```
预期价格 = 10 ^ (5.84 × log₁₀(比特币存活天数) - 17.01)
```

用人话解释：

```mermaid

    title 比特币指数增长预期价格（理论模型）
    2012 : 存活1000天 : 预期价 $10
    2015 : 存活2200天 : 预期价 $400
    2018 : 存活3300天 : 预期价 $4,000
    2021 : 存活4400天 : 预期价 $30,000
    2024 : 存活5500天 : 预期价 $80,000
    2027 : 存活6600天 : 预期价 $180,000
```

> **注意**：这只是数学模型，不是预测！但它提供了一个「如果比特币保持历史增长曲线，价格应该在哪」的参考。

## 4. Ahr999 计算公式

```mermaid
flowchart LR
    P[市场价格 P] --> D1
    M[200日均线 M] --> D1
    D1["比率1 = P ÷ M"] 

    P --> D2
    E[预期价格 E] --> D2
    D2["比率2 = P ÷ E"]

    D1 & D2 --> RESULT["Ahr999 = 比率1 × 比率2\n= (P/M) × (P/E)"]
```

**数学含义**：
- `比率1 < 1`：价格低于200日均线（相对历史偏低）
- `比率2 < 1`：价格低于模型预期（便宜）
- 两个都小于1 → Ahr999 会很小 → 绝佳买入机会

## 7. 完整计算流程

```mermaid
sequenceDiagram
    participant System as 系统
    participant CG as CoinGecko API
    participant Engine as Ahr999 引擎

    System ->> CG: 请求当前 BTC 价格
    System ->> CG: 请求过去200天的历史价格
    CG -->> Engine: 返回价格数据

    Engine ->> Engine: 计算200日均线（简单平均）
    Engine ->> Engine: 计算比特币存活天数（从2009年1月3日起）
    Engine ->> Engine: 代入公式计算「预期价格」
    Engine ->> Engine: 计算 Ahr999 = (价格/200均线) × (价格/预期价格)
    Engine ->> Engine: 判断区间（<0.45 / 0.45-1.2 / >1.2）

    Engine -->> System: 返回指数值 + 区间标签 + 完整数据
```

## 8. 本指标的局限性

```mermaid
mindmap
    root[Ahr999 局限性]
        纯数学模型
            不能预测短期价格
            只适合判断中长期估值
        历史规律不等于未来
            监管黑天鹅无法预测
            ETF等新变量影响曲线
        必须配合其他指标使用
            配合 FGI 判断市场情绪
            配合宏观数据判断大环境
```

> **使用建议**：把 Ahr999 作为定投「力度」的参考，不要作为唯一决策依据。比如 Ahr999 < 0.5 时可以双倍加仓；> 1.5 时可以减半定投，而不是停止。
